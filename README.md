# CarbonForge Porsche B2C Frontend

Next.js storefront for classic Porsche parts and bespoke modifications.

## Vercel

- Framework preset: Next.js
- Install command: `pnpm install`
- Build command: `pnpm build`
- Output directory: Next.js default
- Node.js: 20 or newer

The deployable GLB files are in `public/models`, so Vercel serves them as static assets:

- `/models/porsche-911-2017-gt3-rs.glb`
- `/models/porsche-911-2019-gt3-rs.glb`
